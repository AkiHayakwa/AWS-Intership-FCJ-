### Week 8 Objectives:

* Understand the concepts of GraphQL and how AWS AppSync simplifies API creation.
* Learn to build real-time messaging systems and APIs using AWS AppSync.
* Practice integrating AWS AppSync with Amazon DynamoDB and other data sources.
* Master CRUD operations and data querying using GraphQL schemas, resolvers, and queries/mutations.
* Understand data security and privacy using Amazon Macie to discover, monitor, and protect sensitive data in AWS.
* Learn to deploy WordPress to Amazon EC2 and automate application deployments using AWS CodeDeploy.
* Understand virtual desktop infrastructure (VDI) and deployment using Amazon WorkSpaces.

### Week 8 Achievements:

* **AWS AppSync & GraphQL Integration:**
  * Mastered the creation of scalable and responsive GraphQL APIs using AWS AppSync.
  * Designed flexible GraphQL schemas with appropriate Queries, Mutations, and Subscriptions.
  * Connected AppSync resolvers directly to DynamoDB tables to perform secure CRUD operations.

* **Real-time & Offline Synchronization:**
  * Understood the mechanism of real-time updates using GraphQL Subscriptions.
  * Explored offline data synchronization features and fine-grained access control in AWS AppSync.

* **Amazon Macie & Data Security:**
  * Understood how Amazon Macie automates the discovery of sensitive data (PII, financial data) in S3.
  * Learned how to configure sensitive data discovery jobs, custom data identifiers, and allow lists.
  * Explored S3 bucket security monitoring, policy findings, and integration with EventBridge and Security Hub.

* **WordPress Deployment & AWS CodeDeploy:**
  * Deployed WordPress onto a single Amazon EC2 instance running Amazon Linux/RHEL.
  * Configured AWS CodeDeploy to automate software deployments, eliminate manual tasks, and handle updates with minimal downtime.

* **Amazon WorkSpaces & Virtual Desktop Infrastructure:**
  * Explored how to deploy and manage virtual desktops (Windows, macOS, Ubuntu) securely in the cloud.
  * Understood client-based and web-based secure connection options for end-users.
